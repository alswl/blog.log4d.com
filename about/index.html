我常使用的线上服务（推介链接）

  • Toodledo 2010 年使用至今的 GTD 软件，2015 成为付费用户。纯正 GTD 实现，Matrix 自定义搜索，Open API 全功能可用
  • Evernote 2010 年使用至今的笔记软件，Note 软件届的 Big Father。
  • Dropbox 自 2009 年使用至今，小文件云存储服务
  • RescueTime 自 2016 使用至今，帮助记录时间耗用，从而提高效率的记录工具
  • Shadowsocks 自 2015 使用至今，还不错的 XX 工具
  • Sentry 还不错的 Python Log 收集管理工具
  • Qiniu 自 2012 使用至今的云存储，本博客图片由七牛云存储并加速
  • Linode 老牌 VPS，使用超过 3 年
  • DigitalOcean 曾使用 2 年，可能是性价比最高的 VPS
  • 美团云 自 2017 年使用至今的云服务，全量备份数据。美团云的对象存储和 S3 的 API 兼容度是我测试下来最高的
  • V2EX 创意工作者聚集地
